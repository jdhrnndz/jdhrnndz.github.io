  • Chroma Keying Program: This project is a simple program that accepts input data in the form of image sequences and processes it to make a video with a different background. The image are extracted from a video file using ffmpeg. The video must be shot in front of a green cloth. The program will then remove all green pixels and replace the background from another image or video. This program is done using imagelab, a C++ library for image processing.
